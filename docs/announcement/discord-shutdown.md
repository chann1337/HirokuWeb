# Discord 服务器停止运营公告

::: warning 重要公告
这是由 @Movtery 亲自发布的公告，转发自 GitHub。如果你对此有任何异议，请通过 GitHub 联系 Movtery 本人，最终解释权在于他。
:::

## 公告内容

**Movtery:**

由于某些原因，我已无法访问我的 Discord 账号，目前已无法继续对 "ZalithLauncher Discord Server" 进行管理和运营。

在与 Confectionary(@confectionaryuwu)，也就是本服务器目前唯一的 Admin 进行沟通并讨论后，我们决定正式停止本 Discord 服务器的使用。由于服务器拥有者账号无法访问，当前该服务器无法被删除。

因此，所有频道将被隐藏和锁定，不会再进行任何更新或活动。在之后，我们会重新建立一个新的 Discord 服务器，在此期间，请不要再过多打扰 Confectionary(@confectionaryuwu)

感谢大家的理解与支持。
